# 外区 美区 Apple ID 使用方法


注：修改资料后，在 App Store 里登录，不要在设置里登录 iCloud，否则手机被锁后果自负！

## 更改邮箱、密码

1. 电脑网页打开  https://account.apple.com 并登录。若弹出【Apple ID 安全】选择【其他选项】→【不升级】。
2. 在“登录和安全”部分，选择【Apple 账户】更改邮箱，添加自己的邮箱后删除原邮箱。
3. 再返回更改【密码】。
![](https://i.imgur.com/knuXDH2.png)

## 更改密保

1. 打开 https://iforgot.apple.com 点击【重置密码】；
2. 输入邮箱和验证码，点击【继续】；
3. 选择【重设安全提示问题】，点击【继续】；
4. 回答原密保一个问题，然后即可重设密保。

>注：更改密保后，建议再开启双重认证（绑定手机号），账号更安全。

## 登录

1. 打开 App Store 点击右上角【头像】，下滑至底部点击【退出登录】；
2. 输入新 Apple ID 账号和密码，点击【登录】。

## 提示“无法登录”

开飞行模式，3分钟后关闭，用 WI-FI 登录。如果不行用手机流量登录，再不行重启手机多次尝试几遍。

## 更新 App 时提示“锁定？禁用等信息”

把 App 卸载重装。或者该 App 是别的 Apple ID 下载的，请使用原 ID 密码。

## 禁用锁定

打开 https://iforgot.apple.com 根据密保重设密码。
