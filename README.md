# 外区 美区 Apple ID 使用方法

‼️注：App Store 商店下载专用，请勿在设置里登录 iCloud

## 一、更改邮箱、密码

1. 电脑浏览器打开  https://account.apple.com 并登录。\
	注：若弹出【Apple ID 安全】，选择【其他选项】→【不升级】。
2. 在“登录和安全”部分，选择【电子邮箱】添加自己的邮箱，再删除原邮箱。
3. 返回，再改【密码】。
![](https://i.imgur.com/hHLQVDk.png)

## 二、更改密保

1. 打开 https://iforgot.apple.com 点击【重置密码】；
2. 输入邮箱和验证码，点击【继续】；
3. 选择【重设安全提示问题】，点击【继续】；
4. 回答原密保一个问题，然后即可重设密保。

‼️注：更改密保后，建议开启双重认证（即绑定手机号），账号更安全。

## 三、登录 App Store

1. iPhone 设置，点击头像 → 【媒体与购买项目】，退出登录
2. 登录新账号

‼️注：设置里不要登录 iCloud，仅在【媒体与购买项目（App Store）】登录使用。

## 四、提示“无法登录”

开飞行模式，3 分钟后关闭，用 WI-FI 登录。若不行用手机流量登录，再不行重启手机多次尝试几遍。

## 五、禁用锁定

打开 https://iforgot.apple.com 根据密保重设密码。

## 六、官方客服

遇到任何问题请联系苹果官方客服，英文好的用户直接联系对应国家客服，或者先联系中国客服要求转接国际客服，一般是转到新加坡（中文）。
